# FitLife App

## Project Name
FitLife App

## Description
FitLife is a modern fitness tracking application designed to help users discover and manage their workouts. The app follows the **Layered Architecture** pattern, ensuring a strict separation of concerns among models, services, state management (providers), and user interface components (screens and widgets). This promotes maintainability, scalability, and easier testing.

## Prerequisites & Setup
Make sure you have Flutter installed on your machine.
- **Flutter SDK**: >=3.0.0

To run the application, follow these steps:
1. Clone the repository and navigate to the project directory.
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the automated tests to ensure everything is working correctly:
   ```bash
   flutter test
   ```
4. Start the app on your preferred device or emulator:
   ```bash
   flutter run
   ```

## Dependencies
| Package | Description |
|---|---|
| `provider` | Used for robust, clean State Management across the app (workouts, theme mode). |
| `shared_preferences` | Used to persist the user's theme choice locally across app restarts. |
